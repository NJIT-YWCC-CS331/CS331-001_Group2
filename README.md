SYSTEM NOTES: 
Ubuntu 25.10
Case of using XAMPP: Index and other views must be in ~/opt/lampp/htdocs/

REQUIREMENTS:
PHP 8.2.12 (Bundled with XAMPP)
XAMPP for Linux 8.2.12-0

THIRD PARTY LIBRARIES:
This project uses LM PHP Datagrid for CRUD (Create, Read, Update, Delete) operations and formatting, written by Ian Soko.
[github: https://github.com/lazymofo/datagrid?tab=readme-ov-file]

TO RUN THIS APP IN UBUNTU: 
1] start LAMPP 
2] open browser to localhost//opt/lampp/htdocs/[REST OF EXTENSION]

SECURITY NOTES: 
Do not use user-registration logic for applications where data security is critical.
